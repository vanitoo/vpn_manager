# Changelog

## v3.2.0 — Screen Banners

- Добавлены баннеры экранов: главное меню, книжная полка, об авторе, поддержка.
- Баннеры редактируются в админке через раздел `🖼 Баннеры`.
- Кнопка поддержки теперь открывает экран поддержки с баннером и кнопкой перехода.
- Добавлена настройка `ADMIN_NOTIFY_PURCHASES` для уведомлений админам о продажах.


## v3.1.1

- Исправлено сохранение контакта для чеков: email/телефон теперь сохраняется при первом платеже и используется для следующих покупок через ЮKassa.
- Добавлены диагностические логи по контактам чеков.


## v3.1.0 — Pending Payments UX

- Добавлены ожидающие внешние платежи на карточке книги.
- При возврате из ЮKassa по deep-link бот автоматически проверяет оплату.
- Кнопка `✅ Проверить оплату` показывается в карточке книги до истечения TTL.
- Добавлена настройка `PENDING_PAYMENT_TTL_MINUTES`.


## v3.0.1

- Исправлен старт с несколькими прокси в режимах `sticky`, `rotate`, `random`: перед созданием сессии бот проверяет прокси и выбирает живой.
- Если primary прокси не отвечает, запуск не должен падать на `deleteWebhook`, а должен перейти к следующему доступному прокси.

# Changelog

## v3.0.0 — Cover Processing

- Добавлен модуль `app/covers.py`.
- Обложки теперь автоматически приводятся к формату `cover.jpg`.
- Создается миниатюра `cover_small.jpg`.
- Поддерживаются JPG, PNG, WEBP.
- Добавлены настройки `COVER_WIDTH`, `COVER_HEIGHT`, `COVER_QUALITY`, `GENERATE_COVER_THUMBNAIL`.

## v2.9.0 — Version & System Info

- Добавлен единый источник версии: `app/version.py`.
- Добавлены `APP_NAME`, `APP_VERSION`, `BUILD_DATE`, `DB_SCHEMA_VERSION`.
- При старте в логах выводится версия бота, build date, схема базы, статистика, платежки, режим чеков и лог-файл.
- В админке добавлен пункт `ℹ️ О системе`.
- В SQLite добавлена служебная таблица `app_meta` для хранения версии схемы базы.

## v2.8.0 — Receipts
